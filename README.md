# Pacific Aerials - Drone Photography & Videography

A modern, professional website for Pacific Aerials, showcasing drone photography and cinematic videography services in Fiji.

## Features

- **Responsive Design** - Fully optimized for desktop and mobile devices
- **Hero Video Section** - Stunning full-screen video background
- **Service Cards** - Showcase all drone services offered
- **Portfolio Gallery** - Display video work and projects
- **Smooth Animations** - Fade-in animations as you scroll
- **Contact Section** - Easy booking and contact information
- **Modern Styling** - Dark theme with gold accents

## Project Structure

```
pacific-aerials/
```

## Getting Started

1. Clone the repository:
```bash
git clone https://github.com/aymaanraza83-hash/pacific-aerials.git
cd pacific-aerials
```

2. Add your media files:
   - Place your logo in `images/logo.png`
   - Place your videos in the `videos/` directory

3. Open `index.html` in your browser to view the website locally

## Deployment with GitHub Pages

1. Go to your repository settings
2. Navigate to **Pages** section
3. Select **Deploy from a branch**
4. Choose `main` (or your default branch) as the source
5. Your site will be live at `https://aymaanraza83-hash.github.io/pacific-aerials/`

## Customization

- **Colors**: Edit the CSS in `style.css` (currently using gold `#FFD700`)
- **Contact Info**: Update the contact section in `index.html`
- **Services**: Modify or add service cards in the services section
- **Videos**: Replace video files with your own aerial footage

## Technologies Used

- HTML5
- CSS3 (with media queries for responsiveness)
- JavaScript (for smooth animations)
- Google Fonts (Poppins)
